# ft_printf

Ft_printf is a School 42 project.
The purpose of this project is to create own C library.

[`ft_printf.en.pdf`](/ft_printf.en.pdf) is the task file.

## Usage

To compile the project, run:

```shell
$ make
```

To test the program:

```shell
$ make ex
$ ./a.out
```
